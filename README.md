## Dependencies

### Backend

To install the backend dependencies, navigate to the backend directory and run:

`yarn install`

To see available options, run:
```
sh a_launcher.sh
```

### Frontend

To install the frontend dependencies, navigate to the frontend directory and run:

`npm install`
